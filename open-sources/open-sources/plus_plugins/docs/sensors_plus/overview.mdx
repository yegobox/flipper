## Sensors Plus Overview

A Flutter plugin to access the accelerometer, gyroscope and magnetometer sensors.

### Get started

#### 1. Add it to your project

```yaml {4} title="pubspec.yaml"
dependencies:
  flutter:
    sdk: flutter
  sensors_plus: "^{{ plugins.sensors_plus }}"
```

#### 2. Download dependency

```bash
$ flutter pub get
```

#### 3. Rebuild the app

```bash
$ flutter run
```
