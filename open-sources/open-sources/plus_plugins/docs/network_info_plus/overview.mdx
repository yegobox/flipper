## Network Info Plus Overview

This plugin allows Flutter apps to discover network info and configure
themselves accordingly.

### Get started

#### 1. Add it to your project

```yaml {4} title="pubspec.yaml"
dependencies:
  flutter:
    sdk: flutter
  network_info_plus: "^{{ plugins.network_info_plus }}"
```

#### 2. Download dependency

```bash
$ flutter pub get
```

#### 3. Rebuild the app

```bash
$ flutter run
```
