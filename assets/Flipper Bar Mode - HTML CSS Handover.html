01 — Overview

What Bar Mode is

Standard Flipper POS is a one-cashier, one-cart checkout. Bar Mode turns the same register into a shared bar terminal. It changes three behaviours:

  • Tab per table. Instead of a single anonymous cart, the register keeps a persistent running tab for every table on the floor. Items are added to a table, not to "the sale".
  • Shared register, PIN hand-off. Any staff member logs in with a 4-digit PIN, adds a round to a table, then logs out. The next cashier logs in with their PIN and keeps adding to the same tab. Every line records who logged it.
  • Manager-gated settlement. Servers build the tab; only a manager PIN can take payment and close the table.

The whole feature is off by default and only activates when an admin turns on Table Service (Bar Mode) in the Management Dashboard. When off, the register behaves exactly as it does today.

The core loop. Admin enables Bar Mode → cashier enters PIN → picks a table → logs drinks (each line tagged to them) → logs out → next cashier enters PIN → adds to the same tab → … → a manager settles the bill and frees the table.

The five screens

1 · Admin configManagement Dashboard section with the master Bar Mode toggle + sub-settings. Where the behaviour is switched on.
2 · PIN lockShared-register lock. Pick your name, enter your 4-digit PIN. Shown at start and after every logout.
3 · Table floorZoned floor plan of table cards (open vs free, running total, servers, time open). Entry point after login.
4 · Table POSThe existing POS register, bound to one table. Catalog on the left, the table's running tab on the right with per-line server tags.
5 · Settle billManager-only. Itemised bill grouped by server, VAT breakdown, cash / mobile money, close table.

Plus two overlays: a Manager PIN modal (elevation prompt when a non-manager taps Settle) and a toast (settle confirmation).

The admin gate

The whole feature keys off one boolean, settings.barMode, set in the Admin config screen. Your real app should:

  1. Add the Table Service (Bar Mode) section (§Admin config) to the existing Management Dashboard.
  2. When barMode is on, route the POS entry to the PIN lock instead of the normal register; on login, land on the Table floor; opening a table binds the register to that table.
  3. When barMode is off, do nothing new — the register opens as today.

The four sub-settings refine the flow (see §Store actions for exactly what each one changes): requirePin, floorFirst, managerSettle, autoLogout.

Tab (the running bill per table)

The heart of Bar Mode. tabs is a map of tableId → tab. A table is "open" when a tab exists with ≥1 line; "free" otherwise.

tabs: {
  "T3": {
    openedById: "vm",           // who opened the table
    openedAtMs: 1720000000000,  // epoch ms — drives "34m open" + "Opened 8:12"
    lines: [
      { id:"l1", productId:51, qty:2, price:6500, cashierId:"vm", ts: ... },
      { id:"l2", productId:160, qty:3, price:2400, cashierId:"vm", ts: ... }
    ]
  }
}

Each line carries its own cashierId — that is what powers per-line server tags and the grouped settle bill. A line's price defaults to the product price but can be overridden (shows an "edited" flag).

Derived helpers

  • tabTotal(tab) — Σ price × qty.
  • tabCount(tab) — Σ qty (item count).
  • tabServers(tab) — distinct cashierIds in first-seen order (for the stacked avatars + "logged by N staff").

06 — Store actions

Exact behaviour

The reducer lives in useBarStore() (full source in §data). Each action and what it does:

ActionBehaviour
setScreen(s)Navigate.
setSetting(k,v)Toggle an admin setting.
login(id)Set activeCashierId, go to tables.
logout()Clear cashier + table, go to lock. Tabs are untouched (persist).
openTable(id)Create a tab if none exists (openedById= current cashier, openedAtMs= now), bind activeTableId, go to pos.
addItem(product)Add to the active table's tab. Merges into the current cashier's existing line for that product at the default price (qty++, capped at stock); otherwise pushes a new line tagged with the current cashierId. Another cashier's line for the same product stays separate.
setLineQty(id,n)Clamp 0…stock; a line dropping to 0 is removed.
setLinePrice(id,p)Override unit price (reset restores product price).
delLine(id)Remove a line.
saveToTab()Unbind table, go to tables; if autoLogout also clear cashier and go to lock. (Lines are already live in the tab — this is just "done for now".)
goSettle()Go to settle.
elevateSettle(mgrId)Set active cashier to the approving manager, go to settle. Called from the Manager PIN modal.
settleConfirm()Delete the tab, unbind table, go to tables. (Fire your real payment + receipt here.)
resetDemo()Clear storage + reseed. (Prototype affordance; drop or repurpose in production.)
Add-item merge rule. The dedupe key is productId + cashierId + price===defaultPrice. This keeps each cashier's contribution visible while consolidating repeat taps, and never merges a price-edited line back into a default one.

07 — Persistence

localStorage, session-fresh

State is written to localStorage["flipper-bar-mode-v1"] on every change. On load it rehydrates tabs + settings, but always starts a fresh session: screen:"admin", activeCashierId:null, activeTableId:null. So a refresh never lands mid-settle, and no one stays "logged in" across a reload — matching how a real shared terminal should behave. Line-id counter is restored above the max persisted id to avoid collisions.

Production note. Tabs, staff sessions and settlement should live server-side (multi-device sync, audit). localStorage here is a single-terminal stand-in. Keep the "fresh session on load" rule regardless of backend.

12 — Acceptance checklist

Done means
  • Admin can toggle Bar Mode + the four sub-settings; state persists.
  • With Bar Mode on, the POS entry is the PIN lock; wrong PIN shakes; correct PIN logs the cashier in.
  • Login lands on the table floor (when floorFirst); floor shows open vs free with live totals, servers, and time.
  • Opening a free table creates a tab; opening an open table resumes it.
  • Adding items tags each line to the current cashier; own lines editable, others' read-only (managers edit all).

function useBarStore() {
  const [state, setState] = React.useState(() => {
    try {
      const raw = localStorage.getItem(BAR_STORE_KEY);
      if (raw) {
        const parsed = JSON.parse(raw);
        // reset lineid counter above any persisted ids
        let max = 0;
        Object.values(parsed.tabs || {}).forEach((t) =>
          (t.lines || []).forEach((l) => { const n = parseInt(String(l.id).slice(1), 10); if (n > max) max = n; }));
        __lid = max + 1;
        // persist tabs + settings, but always start a fresh session logged-out
        // at the admin entry so a reload never lands mid-flow.
        return { ...parsed, screen: 'admin', activeCashierId: null, activeTableId: null };
      }
    } catch (e) {}
    return seedState();
  });

  React.useEffect(() => {
    try { localStorage.setItem(BAR_STORE_KEY, JSON.stringify(state)); } catch (e) {}
  }, [state]);

  const patch = (p) => setState((s) => ({ ...s, ...(typeof p === 'function' ? p(s) : p) }));

  const actions = React.useMemo(() => ({
    setScreen: (screen) => patch({ screen }),
    setSetting: (key, val) => setState((s) => ({ ...s, settings: { ...s.settings, [key]: val } })),

    login: (cashierId) => setState((s) => ({
      ...s, activeCashierId: cashierId,
      screen: s.settings.floorFirst ? 'tables' : 'tables',
    })),
    logout: () => patch({ activeCashierId: null, activeTableId: null, screen: 'lock' }),

    openTable: (tableId) => setState((s) => {
      const tabs = { ...s.tabs };
      if (!tabs[tableId]) {
        tabs[tableId] = { openedById: s.activeCashierId, openedAtMs: Date.now(), lines: [] };
      }
      return { ...s, tabs, activeTableId: tableId, screen: 'pos' };
    }),

    addItem: (product) => setState((s) => {
      const tid = s.activeTableId; if (!tid) return s;
      const tab = s.tabs[tid] || { openedById: s.activeCashierId, openedAtMs: Date.now(), lines: [] };
      const lines = tab.lines.slice();
      // merge into this cashier's existing (unedited-price) line for the product
      const idx = lines.findIndex((l) => l.productId === product.id && l.cashierId === s.activeCashierId && l.price === product.price);
      if (idx >= 0) lines[idx] = { ...lines[idx], qty: Math.min(product.stock, lines[idx].qty + 1) };
      else lines.push({ id: lid(), productId: product.id, qty: 1, price: product.price, cashierId: s.activeCashierId, ts: Date.now() });
      return { ...s, tabs: { ...s.tabs, [tid]: { ...tab, lines } } };
    }),

    setLineQty: (lineId, qty) => setState((s) => {
      const tid = s.activeTableId; const tab = s.tabs[tid]; if (!tab) return s;
      let lines = tab.lines.map((l) => {
        if (l.id !== lineId) return l;
        const p = barProduct(l.productId);
        const n = Math.max(0, Math.min(p ? p.stock : 9999, qty));
        return { ...l, qty: n };
      }).filter((l) => l.qty > 0);
      return { ...s, tabs: { ...s.tabs, [tid]: { ...tab, lines } } };
    }),
    setLinePrice: (lineId, price) => setState((s) => {
      const tid = s.activeTableId; const tab = s.tabs[tid]; if (!tab) return s;
      const lines = tab.lines.map((l) => l.id === lineId ? { ...l, price } : l);
      return { ...s, tabs: { ...s.tabs, [tid]: { ...tab, lines } } };
    }),
    delLine: (lineId) => setState((s) => {
      const tid = s.activeTableId; const tab = s.tabs[tid]; if (!tab) return s;
      return { ...s, tabs: { ...s.tabs, [tid]: { ...tab, lines: tab.lines.filter((l) => l.id !== lineId) } } };
    }),

    saveToTab: () => setState((s) => ({
      ...s, activeTableId: null,
      screen: s.settings.autoLogout ? 'lock' : 'tables',
      activeCashierId: s.settings.autoLogout ? null : s.activeCashierId,
    })),

    goSettle: () => patch({ screen: 'settle' }),
    elevateSettle: (mgrId) => patch({ activeCashierId: mgrId, screen: 'settle' }),
    settleConfirm: () => setState((s) => {
      const tabs = { ...s.tabs }; delete tabs[s.activeTableId];
      return { ...s, tabs, activeTableId: null, screen: 'tables' };
    }),

    resetDemo: () => { try { localStorage.removeItem(BAR_STORE_KEY); } catch (e) {} setState(seedState()); },
  }), []);

  return [state, actions];
}

/* Controlled-ish PIN entry: manages its own value, calls onComplete(pin).
   Parent bumps `errorNonce` to shake + clear after a wrong PIN. */
function PinEntry({ len = 4, onComplete, disabled, errorNonce = 0, keyboard }) {
  const [val, setVal] = useScr('');
  const [err, setErr] = useScr(false);
  const firstErr = useScrR(errorNonce);

  useScrE(() => {
    if (errorNonce !== firstErr.current) {
      firstErr.current = errorNonce;
      setErr(true);
      const t1 = setTimeout(() => { setVal(''); }, 380);
      const t2 = setTimeout(() => setErr(false), 420);
      return () => { clearTimeout(t1); clearTimeout(t2); };
    }
  }, [errorNonce]);

  const push = (d) => {
    if (disabled) return;
    setErr(false);
    setVal((v) => {
      if (v.length >= len) return v;
      const nv = v + d;
      if (nv.length === len) setTimeout(() => onComplete(nv), 90);
      return nv;
    });
  };
  const back = () => { setErr(false); setVal((v) => v.slice(0, -1)); };
  const clear = () => { setErr(false); setVal(''); };

  useScrE(() => {
    if (!keyboard) return;
    const h = (e) => {
      if (disabled) return;
      if (/^[0-9]$/.test(e.key)) push(e.key);
      else if (e.key === 'Backspace') back();
    };
    window.addEventListener('keydown', h);
    return () => window.removeEventListener('keydown', h);
  }, [keyboard, disabled]);

  return (
    <>
      <PinDots len={len} filled={val.length} error={err} />
      <Keypad onDigit={push} onBackspace={back} onClear={clear} disabled={disabled} />
    </>
  );
}

/* ============================================================
   SETTLE — manager pays a table's running bill
   ============================================================ */
function Settle({ store, actions, notify }) {
  const table = barTable(store.activeTableId);
  const tab = store.tabs[store.activeTableId];
  const [method, setMethod] = useScr('cash');
  const [tender, setTender] = useScr('');
  const manager = barCashier(store.activeCashierId);

  if (!table || !tab) return null;

  const total = tabTotal(tab);
  const vat = total - total / 1.18;
  const tenderNum = parseFloat(String(tender).replace(/[^\d.]/g, '')) || 0;
  const change = Math.max(0, tenderNum - total);
  const due = Math.max(0, total - tenderNum);
  const payState = method === 'momo' ? 'ok' : (tenderNum <= 0 ? 'idle' : tenderNum < total ? 'due' : tenderNum === total ? 'exact' : 'change');
  const canPay = method === 'momo' || tenderNum >= total;

  // group lines by server, preserving first-seen order
  const groups = [];
  tab.lines.forEach((l) => {
    let g = groups.find((x) => x.cashierId === l.cashierId);
    if (!g) { g = { cashierId: l.cashierId, lines: [] }; groups.push(g); }
    g.lines.push(l);
  });

  const confirm = () => {
    actions.settleConfirm();
    notify(`${table.id} settled · RWF ${barMoney(total)} ${method === 'momo' ? 'Mobile Money' : 'Cash'}`);
  };

  return (
    <div className="bar-settle bar-fade">
      <div className="bar-settle-top">
        <button className="pos-backbtn" onClick={() => actions.setScreen('pos')}><Icons.ChevLeft size={18} /> Back to tab</button>
        <div className="ttl"><span className="badge">{table.id}</span> Settle bill · {table.zoneName}</div>
        <div className="bar-floor-spacer" />
        {manager && (
          <div className="bar-chip">
            <span className="bar-chip-av" style={{ background: manager.color }}>{manager.initials}</span>
            <span className="bar-chip-meta">
              <span className="bar-chip-name">{manager.name}</span>
              <span className="bar-chip-role">Settling as manager</span>
            </span>
          </div>
        )}
      </div>

      <div className="bar-settle-body">
        <div className="bar-settle-bill">
          <div className="bar-bill-card">
            <div className="bar-bill-head">
              <span className="h">Table {table.id} — running tab</span>
              <span className="m">Opened {barFmtClock(tab.openedAtMs)} · {barFmtElapsed(tab.openedAtMs)}</span>
            </div>
            {groups.map((g) => {
              const c = barCashier(g.cashierId);
              const gt = g.lines.reduce((s, l) => s + l.price * l.qty, 0);
              return (
                <div className="bar-bill-group" key={g.cashierId}>
                  <div className="bar-bill-grouphd">
                    <span className="av" style={{ background: c ? c.color : '#999' }}>{c ? c.initials : '?'}</span>
                    <span className="n">{c ? c.name : 'Unknown'}</span>
                    <span className="r">· {c ? c.role : ''}</span>
                    <div className="bar-floor-spacer" />
                    <span className="r" style={{ fontFamily: 'var(--mono)' }}>RWF {barMoney(gt)}</span>
                  </div>
                  {g.lines.map((l) => {
                    const p = barProduct(l.productId);
                    return (
                      <div className="bar-bill-line" key={l.id}>
                        <span className="q">{l.qty}×</span>
                        <span className="nm">{p ? p.name : 'Item'} <span className="unit">@ {barMoney(l.price)}</span></span>
                        <span className="amt">{barMoney(l.price * l.qty)}</span>
                      </div>
                    );
                  })}
                </div>
              );
            })}
            <div className="bar-bill-foot">
              <div className="bar-bill-row"><span className="k">Subtotal (excl. VAT)</span><span className="v">RWF {barMoney(total - vat)}</span></div>
              <div className="bar-bill-row"><span className="k">VAT 18%</span><span className="v">RWF {barMoney(vat)}</span></div>
              <div className="bar-bill-row grand"><span className="k">Total due</span><span className="v"><small>RWF</small> {barMoney(total)}</span></div>
            </div>
          </div>
        </div>

        <div className="bar-settle-pay">
          <div className="bar-pay-h">Payment</div>
          <div className="bar-pay-sub">Choose method and take payment to close the table.</div>
          <div className="bar-pay-methods">
            <button className={`bar-pay-method ${method === 'cash' ? 'is-on' : ''}`} onClick={() => setMethod('cash')}>
              <span className="ic"><Icons.Wallet size={18} /></span><span className="nm">Cash</span>
            </button>
            <button className={`bar-pay-method ${method === 'momo' ? 'is-on' : ''}`} onClick={() => setMethod('momo')}>
              <span className="ic"><Icons.Phone size={18} /></span><span className="nm">Mobile Money</span>
            </button>
          </div>

          {method === 'cash' && (
            <>
              <div className={`pos-readout ${payState}`} style={{ marginTop: 0 }}>
                <div className="pos-readout-l">
                  <span className="pos-readout-k">
                    {payState === 'idle' ? 'Enter amount tendered' : payState === 'due' ? 'Balance due' : payState === 'exact' ? 'Exact — no change' : 'Change to give'}
                  </span>
                  <span className="pos-readout-sub">{payState === 'idle' ? `RWF ${barMoney(total)} due` : `Tendered RWF ${barMoney(tenderNum)}`}</span>
                </div>
                <span className="pos-readout-v">{payState === 'idle' ? '—' : payState === 'due' ? `RWF ${barMoney(due)}` : `RWF ${barMoney(change)}`}</span>
              </div>
              <div className="pos-tender">
                <div className="pos-tender-field">
                  <input inputMode="decimal" placeholder="0" value={tender} onChange={(e) => setTender(e.target.value)} />
                  <span className="pos-tender-cur">RWF</span>
                </div>
              </div>
              <div className="pos-quickcash">
                {[total, 20000, 50000, 100000].map((v, i) => (
                  <button key={i} className="pos-qc" onClick={() => setTender(String(Math.round(v)))}>{i === 0 ? 'Exact' : barMoney(v)}</button>
                ))}
              </div>
            </>
          )}
          {method === 'momo' && (
            <div style={{ padding: '18px', borderRadius: 'var(--r-md)', background: 'var(--surface-2)', border: '1px solid var(--line)', fontSize: 13.5, color: 'var(--ink-2)', lineHeight: 1.5 }}>
              A payment request for <b>RWF {barMoney(total)}</b> will be pushed to the customer's phone. Confirm once the customer approves.
            </div>
          )}

          <div className="bar-pay-spacer" />
          <button className="bar-btn bar-btn-primary" style={{ width: '100%', height: 58, fontSize: 16 }} disabled={!canPay} onClick={confirm}>
            <Icons.ShieldCheck size={19} /> Confirm payment · RWF {barMoney(total)}
          </button>
          <div className="bar-pos-foot-note" style={{ justifyContent: 'center', marginTop: 12 }}>
            <Icons.ShieldCheck size={13} /> Closing the table saves the sale and frees it for new guests.
          </div>
        </div>
      </div>
    </div>
  );
}

/* ============================================================

   Requires: React, ReactDOM, all bar/*.jsx above.

const { useState: useApp, useEffect: useAppE, useRef: useAppR } = React;

function BarStage({ children }) {
  const ref = useAppR(null);
  useAppE(() => {
    const fit = () => {
      const el = ref.current; if (!el) return;
      const s = Math.min(window.innerWidth / 1440, window.innerHeight / 912);
      el.style.transform = `translate(-50%, -50%) scale(${s})`;
    };
    fit();
    window.addEventListener('resize', fit);
    return () => window.removeEventListener('resize', fit);
  }, []);
  return (
    <div className="bar-stage">
      <div className="bar-canvas" ref={ref}>{children}</div>
    </div>
  );
}

function BarApp() {
  const [store, actions] = useBarStore();
  const [toast, setToast] = useApp('');
  const [mgrModal, setMgrModal] = useApp(false);
  const toastT = useAppR(0);

  const notify = (msg) => {
    setToast(msg);
    clearTimeout(toastT.current);
    toastT.current = setTimeout(() => setToast(''), 3200);
  };

  // gentle clock tick so "34m open" stays fresh while reviewing
  const [, tick] = useApp(0);
  useAppE(() => { const t = setInterval(() => tick((n) => n + 1), 30000); return () => clearInterval(t); }, []);

  const onSettle = () => {
    const c = barCashier(store.activeCashierId);
    if (store.settings.managerSettle && !(c && c.manager)) setMgrModal(true);
    else actions.goSettle();
  };

  let screen = null;
  if (store.screen === 'admin') screen = <AdminConfig store={store} actions={actions} />;
  else if (store.screen === 'lock') screen = <PinLock store={store} actions={actions} />;
  else if (store.screen === 'tables') screen = <TableFloor store={store} actions={actions} />;
  else if (store.screen === 'pos') screen = <BarPOS store={store} actions={actions} onSettle={onSettle} />;
  else if (store.screen === 'settle') screen = <Settle store={store} actions={actions} notify={notify} />;
  else screen = <PinLock store={store} actions={actions} />;

  // guard: if a screen needs an active cashier / table but none, fall back
  if ((store.screen === 'tables' || store.screen === 'pos' || store.screen === 'settle') && !store.activeCashierId) {
    screen = <PinLock store={store} actions={actions} />;
  }
  if ((store.screen === 'pos' || store.screen === 'settle') && !store.activeTableId) {
    screen = <TableFloor store={store} actions={actions} />;
  }

  return (
    <BarStage>
      <div className="bar-screen">{screen}</div>
      <BarToast msg={toast} />
      {mgrModal && (
        <ManagerPinModal
          onCancel={() => setMgrModal(false)}
          onSuccess={(mgrId) => { setMgrModal(false); actions.elevateSettle(mgrId); }}
        />
      )}
    </BarStage>
  );
}
